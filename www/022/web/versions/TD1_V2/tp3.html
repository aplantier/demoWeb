Des boi-boites

Il existe deux catégories de boites exceptions faites des tableaux ou listes

Inline

comportements comme des lettres, les uns à la suite des autres sur un axe horizontal

  • de texte : b, i, em, strong, sup, sub
  • Images img
  • Divers
  • Liens a
  • Balise neutre span
Block

comportements comme des paragraphes, les uns au dessus des autres sur un axe vertical et en occupant toute la largeur de la page par dééfaut (ou du bloc parent )

  • Titres : h1, h2, ..., h6
  • Balises structurant la page : main, section, header, footer, nav, aside...
  • Paragraphes : p
  • Listes : ol, ul
  • Divers : blockquote, pre...
  • Balise neutre : div

Exercice

Config du main

On configure le css pour parametrer l'affichage du main de la page pour que :
- sa largeur soit de 80 % de la page
- Sa largeur max soit de 1000px
- sa largeur min soit de 500px

Le display sur lequel a été testéé ce code est un 27 pouces, le main est donc limité a 1000px de largeur. lorsque l'on modifie la taille de la fenetre, la taille du main s'adapte dynamiquement jusqu'au seuil de 500px ou la il ne diminue pas moins ( et bloque la redimenssion de la page à ce seuil)

main { /** TD3 : Setup du main */ background-color: blue; width: 80%; min-width: 500px; max-width: 1000px; }

Exercice

propriétés racourcies

Soit le code suivant :

h1 { border: 2px red 4px yellow;}

Comment se comportera l'affichage ?
Ne pas tout faire en même temps !!
le code semble vouloir modifier la taille et couleur des bordures Haute et basse, mais nous ne pouvons pas mélanger ces propriétés. deplus il faut définir une propriété de style
Ainsi pour le resultats escompté, privilégier :

h1{ border-width: 4px 2px ; /* épaisseurs bordures top& */ border-color: red yellow; /* couleur bordures top& botbr> border-style: dashed; /* style bordures top& bot */ }

Questions :
  • Quels avantages y a-t-il à utiliser une taille absolue ? Une taille relative ?:

    La taille absolue est plus spécfifique. On peut être très précis mais le rendu sera donc très similaire en fonction du materiel/ navigateur utilisé. En revanche la page sera statique et immuable. a la différence de la taille relative.

    L'unité em correspond à une tabulation. En fonctions des systèmes, une tabulation peut etre composé d'un nombre d'espace différent. Quelle est la nouvelle unité
